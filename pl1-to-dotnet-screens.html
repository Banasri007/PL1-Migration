PL/I Screen to .NET Screen Conversion

A visual comparison of the original CICS/BMS transaction screens and the converted ASP.NET Core MVC screens.

Original PL/I / CICS Screen CUSTSRCH
-------------------------------------------------- CUSTOMER SEARCH -------------------------------------------------- Transaction : CUST Program : CUSTPGM Map : CUSTSRCH Customer ID : ________ PF3=Exit Enter=Search --------------------------------------------------
BMS map field CUSTID-I is received by the PL/I program and used to read the customer record.
Converted .NET Screen Customer/Search
TRANSACTION: CUST SCREEN 1 OF 3

Customer Search

Razor form posts customerId to CustomerController.Details.
Original PL/I / CICS Screen CUSTDTL
-------------------------------------------------- CUSTOMER DETAILS -------------------------------------------------- Customer ID : 10001 Name : JOHN SMITH Status : ACTIVE Balance : 2500.75 New Status : ________ PF3=Back PF5=Update --------------------------------------------------
PL/I populates output fields from DB2/VSAM, then receives NEWSTATUS-I when the user presses PF5.
